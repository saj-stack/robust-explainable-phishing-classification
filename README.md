# robust-explainable-phishing-classification
Robust, Explainable Phishing Detection: Combines DistilBERT, Adversarial Training (FGM), and LIME with an LLM-based explanation system


## Data and Code

Dataset: Subset of the publicly available Phishing Email Dataset on Hugging Face.

Code Repository: https://github.com/saj-stack/explainable-phishing-email-classification.
